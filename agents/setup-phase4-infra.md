## Phase 4 인프라 생성 에이전트

**입력**: `{requirements_md_path, generated_agents_list}`

**목표**: 메모리 시드 파일과 `.claude/settings.json`을 생성하고, `setup-phase4-files`의 결과와 합산하여 Phase 4 전체 완료 보고서를 반환합니다.

---

### Step 0: requirements.md 로드

`requirements_md_path`의 파일을 전체 읽어 아래 항목을 파악합니다:
- 프로젝트명, 아키텍처 결정 사항, 코드 컨벤션
- `Permissions`, `Hooks` 섹션
- `Memory Seeds` 섹션 (추가 시드 항목)
- `Agents Needed` 목록 (settings.json hook 생성에 필요)
- 보안 관련 항목 존재 여부 (인증·인가·API·결제·사용자 데이터)

---

### Step 4: ~/.claude/projects/<encoded-path>/memory/ 생성

```bash
# 경로 인코딩: / → - 치환
ENCODED_PATH=$(echo "$PWD" | sed 's|/|-|g')
MEMORY_DIR="$HOME/.claude/projects/${ENCODED_PATH}/memory"
```

`agents/references/memory-templates.md`를 읽고, `docs/requirements.md` 내용으로 각 템플릿의 `<...>` 부분을 채워 아래 5개 파일을 생성합니다:
- `MEMORY.md` — 인덱스 (세션마다 자동 로드)
- `session-log.md` — 작업 완료 로그
- `architecture.md` — 아키텍처 결정 사항 (`docs/requirements.md`의 아키텍처 섹션으로 채움)
- `feedback-conventions.md` — 코드 컨벤션 (`docs/requirements.md`의 코드 스타일 섹션으로 채움)
- `debugging.md` — 디버깅 기록 (빈 파일로 시작)

**추가 Memory Seeds 처리**: `docs/requirements.md`의 `Memory Seeds` 항목에 추가 seed가 있으면 위 기본 파일들에 병합하여 생성합니다.

---

### Step 5: .claude/settings.json 생성

`agents/references/hook-examples.md`를 읽고, 아래 조건에 따라 포함할 훅을 선택하여 `docs/requirements.md`의 `Permissions`·`Hooks` 섹션과 병합합니다.

**포함 조건**:
- **Stop Hook**: 항상 포함
- **PostToolUse Security**: `docs/requirements.md`에 인증·인가·API·결제·사용자 데이터 관련 항목이 있는 경우
- **PostToolUse backend-developer**: `Agents Needed`에 `backend-developer`가 있는 경우
- **PostToolUse frontend-developer**: `Agents Needed`에 `frontend-developer`가 있는 경우
- **PostToolUse ml-engineer**: `Agents Needed`에 `ml-engineer`가 있는 경우

각 훅의 정확한 JSON은 `agents/references/hook-examples.md`의 해당 섹션을 그대로 사용합니다.

---

### 완료: 전체 Phase 4 파일 목록 반환

게이트 메시지 없이, `generated_agents_list`(setup-phase4-files 결과)와 이 에이전트가 생성한 파일들을 합산하여 전체 목록을 출력하고 종료합니다:

```
## Phase 4 완료 — 전체 생성 파일

| 파일 | 설명 |
|------|------|
| CLAUDE.md | 프로젝트 가이드 + Quality Gates (setup-phase4-files) |
| .claude/agents/code-reviewer.md | 코드 품질 리뷰 에이전트 (setup-phase4-files) |
| ... (setup-phase4-files 생성 파일) | ... |
| ~/.claude/projects/<encoded>/memory/MEMORY.md | 메모리 인덱스 |
| ~/.claude/projects/<encoded>/memory/session-log.md | 세션 로그 |
| ~/.claude/projects/<encoded>/memory/architecture.md | 아키텍처 결정 사항 |
| ~/.claude/projects/<encoded>/memory/feedback-conventions.md | 컨벤션 피드백 |
| ~/.claude/projects/<encoded>/memory/debugging.md | 디버깅 기록 |
| .claude/settings.json | 훅·권한 설정 |
```

이 목록을 오케스트레이터(`commands/run.md`)에 전달합니다. 오케스트레이터가 Phase 4→5 게이트 메시지를 표시합니다.
